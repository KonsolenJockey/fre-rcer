Contains the base classes of the Repository Object Model (ROM). The ROM is a set of EMF Ecore models that represent repository objects from the SAP R/3 Development Workbench. The Ecore models can be found in the models directory of the main ROM plug-in. This package and its sub-packages contain the generated implementation classes and the additional classes to read the repository objects using RFC calls.
