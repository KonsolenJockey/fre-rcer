Contains the connection handling mechanism of the connector. The classes in this package deal with the connection data, the login credentials and the actual connections. The central class of this package is the {@link ConnectionManager} - use this class to obtain connections. If you want to provide connection data or login credentials, take a look at the extension points provided by the plug-in.
