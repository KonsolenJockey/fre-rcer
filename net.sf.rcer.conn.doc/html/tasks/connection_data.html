Connection Data

The connector provides a preference page to edit the settings used to connecto to the SAP R/3 system. Every connection requires a description and the system ID. It is possible to add multiple connections with the same system ID.

A connection is either a direct connection to a dispatcher process or a load-balanced connection to a message server. The settings required differ depending on the connection type.

The connection data is saved in the workspace metadata of the application, using the standard preferences mechanism.

Direct Connections

To connect to a SAP R/3 system directly (without using a message server), you have to specify the application server and the system number. Depending on your network setup, you may also have to set a router string.

Load-Balanced Connections

To use a message server, you have to specify the message server name and port and the name of the logon group. Depending on your network setup, you may also have to set a router string.

Default Logon Settings

You can optionally specify some default data to preset the login dialog.
